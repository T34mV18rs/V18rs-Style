## Installation Guide
`Commands` For Termux

```
$ apt update && apt upgrade 
$ pkg install python
$ pkg install python2
$ pkg install gitclone
$ git clone https://github.com/T34mV18rs/V18rs-Style
$ ls
$ cd V18rs-Style
$ chmod +x *
$ bash v18rs-style.sh

```
